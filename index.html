File Launcher V2.4 🚀

Pardus XFCE üzerinde .exe, .jar ve .zip dosyalarını sağ tıklayarak saniyeler içinde çalıştırın!

Proje Hakkında

Pardus Linux işletim sisteminde Windows (.exe) veya Java (.jar) uygulamalarını terminale kod yazmadan, doğrudan çift tıklayarak veya sağ tıklayıp çalıştırabilmenizi sağlayan, Tkinter arayüzlü hafif ve hızlı bir başlatıcı yardımcı programıdır.

Adım Adım Kurulum Kılavuzu 🛠️

File Launcher'ın sisteminizde sorunsuz çalışabilmesi için gerekli tüm adımları aşağıda bulabilirsiniz.

1. Adım: Gerekli Tüm Sistem Araçlarını Kurun

File Launcher arayüzünün açılması, .exe dosyalarının Wine ile çalışması ve .jar (Java) dosyalarının tetiklenmesi için gerekli olan tüm paketleri tek bir komutla kurabilirsiniz. Terminalinizi açın ve aşağıdaki komutu yapıştırıp Enter'a basın:

sudo apt update && sudo apt install python3-tk wine default-jre -y

(Bu komut; Python arayüz motorunu, Windows uygulamaları için Wine'ı ve Java tabanlı Minecraft SKlauncher gibi araçlar için Java motorunu sisteminize otomatik kurar.)

2. Adım: Programı İndirin

Aşağıdaki butona tıklayarak GitHub sayfamıza gidin, yeşil "Code" butonuna basıp "Download ZIP" diyerek projeyi bilgisayarınıza indirin. ZIP içerisindeki File Launcher V2.4 (for Pardus).py dosyasını Masaüstüne çıkartın.

3. Adım: Sağ Tık Entegrasyonunu Sağlayın (Masaüstünüze Göre Seçin)

Herhangi bir exe veya jar dosyasına sağ tıkladığınızda programımızın görünebilmesi için kullandığınız masaüstü ortamına uygun adımı uygulayın:


A Seçeneği: Pardus GNOME (Pardus 25) Kullananlar İçin

GNOME dosya yöneticisinde sağ tık eylemini tanımlamak için sistem genelinde bir kısayol oluşturmamız gerekir:

  1. Terminalinizi açın ve şu komutu yazarak boş bir kısayol dosyası oluşturun:
    2. nano ~/.local/share/applications/file-launcher.desktop
  2. Açılan pencereye aşağıdaki kodları aynen yapıştırın (Kullanıcı adınızı kendi sisteminize göre değiştirmeyi unutmayın):
    3. [Desktop Entry]
Type=Application
Name=File Launcher V2.4
Exec=python3 "/home/KULLANICI_ADIN/Masaüstü/File Launcher V2.4 (for Pardus).py" %f
Icon=system-run
MimeType=application/x-ms-dos-executable;application/x-executable;application/java-archive;application/zip;
NoDisplay=false
Terminal=false
  3. Ctrl + O yapıp Enter ile kaydedin, ardından Ctrl + X ile çıkın. Artık sağ tık eyleminiz GNOME listesinde hazır!

B Seçeneği: Pardus XFCE Kullananlar İçin

XFCE'nin Thunar dosya yöneticisinde sağ tık eylemini tanımlamak için:

  1. Thunar Dosya Yöneticisini açıp üst menüden sırasıyla "Düzenle" ➔ "Özel Eylemleri Yapılandır..." seçeneğine tıklayın.
  2. Sağdaki + (Artı) butonuna basarak yeni bir eylem ekleyin.
  3. Temel sekmesinde; Adı File Launcher ile Aç yapın, komut kısmına ise şu kodu yazın (Kullanıcı adınızı değiştirmeyi unutmayın):
    4. python3 "/home/KULLANICI_ADIN/Masaüstü/File Launcher V2.4 (for Pardus).py" %f
  4. Görünüm Koşulları sekmesine geçip Dosya Deseni kutusuna *.exe;*.jar;*.zip yazın ve alt taraftan "Diğer Dosyalar" seçeneğini işaretleyip kaydedin.

Özellikler

  • Hızlı Çalıştırma: Windows uygulamalarını otomatik olarak Wine ile eşleştirir.
  • Java Desteği: Sisteminizdeki tüm .jar (Java) dosyalarını saniyeler içinde açar.
  • Şık Arayüz: Arka planda hangi uygulamanın başladığını gösteren modern, sade ve hızlı durum ekranı.
