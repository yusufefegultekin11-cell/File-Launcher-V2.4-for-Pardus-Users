File Launcher V2.4 🚀

Pardus XFCE üzerinde .exe, .jar ve .zip dosyalarını sağ tıklayarak saniyeler içinde çalıştırın!

Proje Hakkında

Pardus Linux işletim sisteminde Windows (.exe) veya Java (.jar) uygulamalarını terminale kod yazmadan, doğrudan çift tıklayarak veya sağ tıklayıp çalıştırabilmenizi sağlayan, Tkinter arayüzlü hafif ve hızlı bir başlatıcı yardımcı programıdır.

Nasıl Kurulur? (Adım Adım)

1. Gerekli motoru kurun: Terminali açıp arayüz motorunu sisteme yükleyin:

sudo apt update && sudo apt install python3-tk -y

2. Programı indirin: Projeyi GitHub reposundan masaüstünüze çekin ve sağ tık entegrasyonunu tamamlayın.

Özellikler

  • Hızlı Çalıştırma: Windows uygulamalarını otomatik olarak Wine ile eşleştirir.
  • Java Desteği: Sisteminizdeki tüm .jar (Java) dosyalarını saniyeler içinde açar.
  • Şık Arayüz: Arka planda hangi uygulamanın başladığını gösteren modern, sade ve hızlı durum ekranı.
